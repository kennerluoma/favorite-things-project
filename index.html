Favorite Things

This is a list of five of my favorite things. Click on a button to see what some of my favorite things are. Click this to see how it works.

How it works:

There are two main functions for this page: getThing and button. One gets the data from a JSON file and builds the html structure for each object. The other turns the anchor tags from the first function into buttons to show and hide each favorite thing category.

The getThing function uses a for...of loop to get each JSON object. The variable id uses the category name to assign a unique ID to each article. If there is an image value, bg is assigned a background image property using the image. A button for each JSON object is appended to the #intro section, an article for each is appended to the #things section, and two sections are appended to each of those articles. The first section gets a background image from bg and a title from the name value. The second section contains the description value of the object.

The second function, button, looks for anchor tags with an ID created from the getThing function. It uses a regular expression to match a button's ID with an article's ID. On click it 1: toggles .active on the button, 2: removes .active from other buttons, 3: toggles .hidden on the matching article, and 4: adds .hidden to other articles.
